### Github process
- git checkout main
- git pull --rebase origin main
- git checkout -b feature/task
- git add . && git commit -m "message"
- git push origin feature/task
-> Create PR
-> Review
-> Merge

### Test PR
- git fetch origin
- git checkout feature/task
- Test locally
- Give approval
